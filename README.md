## 📖 Sobre o Repositório

Este repositório reúne todo o material produzido durante meu curso de **Python** na plataforma [Alura](https://www.alura.com.br/), incluindo exercícios resolvidos, projetos práticos e anotações de estudo.

## ▶️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/python-alura.git
   ```

2. Acesse a pasta desejada:
   ```bash
   cd python-alura/exercicios
   ```

3. Execute o arquivo Python:
   ```bash
   python nome_do_arquivo.py
   ```

> Certifique-se de ter o [Python](https://www.python.org/downloads/) instalado na sua máquina.
