# Boas Práticas

Adotar boas práticas em SQL é fundamental não apenas para garatir que as consultas sejam executadas com eficiência, mas também facilitar a leitura, a segurança, a manutenção e a colaboração com outros desenvolvedores.

Aqui estão listadas algumas boas práticas para desenvolver bancos de dados:
- Sempre usar '?' para passar parâmetros.
- Sempre usar 'IF NOT EXISTS' ao criar tabelas.
- Sempre se lembrar de usar o conn.commit() para salvar as alterações.
- Sempre se lembrar de usar o conn.close() para encerrar a conexão com o banco de dados.
