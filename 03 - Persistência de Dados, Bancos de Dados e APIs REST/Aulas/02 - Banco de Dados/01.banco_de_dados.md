# Banco de Dados

Um banco de dados é uma ferramenta que permite o armazenamento, o gerenciamento e a recuperação de dados.

Os bancos de dados são divididos em três tipos:
1. Relacionais (SQL)
2. Não relacionais (NoSQL)
3. Em nuvem (DBaaS)

## Bancos Relacionais (SQL)

Bancos relacionais são aqueles que armazenam os dados em tabelas (linhas e colunas). Eles utilizam a linguagem SQL (Structured Query Language) para consultar e manipular os dados.

Alguns exemplos de bancos relacionais:
- MySQL
- PostgreSQL
- Oracle
- SQL Server

Os bancos relacionais são melhores para: Sistemas financeiros, ERPs, sistemas de inventário e aplicações onde a precisão e a relação entre os dados são cruciais.

## Bancos Não Relacionais (NoSQL)

Bancos não relacionais são aqueles criados para lidar com grande volumes de dados não-estruturados ou semi-estruturados. Bancos NoSQL oferecem esquemas flexiveis e são altamente escaláveis. Eles se dividem em quatro categorias principais:
1. Orientados a documentos
    - Ideais para catálogos de produtos, gerenciamento de conteúdo (CMS) e perfis de usuários
    - Exemplos populares: MongoDB, Couchbase, Firestore
2. Chave-Valor
    - Ideais para caching (armazenamento temporário rápido), gerenciamento de sessões de usuários e carrinhos de compras em e-commerces.
    - Exemplos populares: Redis, Amazon DynamoDB, Memcached
3. Orientados a colunas (Wide-Column)
    - Ideais para análise de Big Data, registro de eventos (logs) e dados de sensores (IoT)
    - Exemplos populares: Apache Cassandra, HBase
4. Baseados em grafos
    - Ideais para redes sociais, mecanismos de recomendação e detecção de fraudes
    - Exemplos populares: Neo4j, Amazon Neptune
